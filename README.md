# BrightLook (MERN)

## Backend (`server/`)
- Express (ES modules)
- CORS enabled
- MongoDB Atlas connection via `mongoose` + `dotenv`
- Runs on `http://localhost:5000`

1. Create `server/.env` from `server/.env.example` and set `MONGODB_URI`
2. Run:
   - `cd server`
   - `npm run dev`

## Frontend (`client/`)
- React + Vite
- Installed: `axios`, `react-router-dom`, `react-hot-toast`, `lucide-react`

Run:
- `cd client`
- `npm run dev`
